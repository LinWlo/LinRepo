# 1. markdown语法
## 1.1 链接引用
### 1.1.1 外部链接
- 语法：
  - `[显示点击的名称](网址)`
  - 例：点这里访问 [百度](http://www.baidu.com)；
  - 注：需要完整网址，从协议开始。
### 1.1.2 文内链接引用
- markdown语法：`[显示点击的名称](#标题名)`
  - 例：点这里到达[外部链接](#外部链接)；
  - 注：只适用于链接到标题处，正文不适用。
- html语法：既可以到标题也可以到正文
  1. 在跳转到的内容处设置锚点：
     - `<span id='名称'>跳转到的正文内容</span>` 
  2. 在需要跳转处设置跳转点：
     - `<a href='#名称'>点击处的正文内容</a>`
